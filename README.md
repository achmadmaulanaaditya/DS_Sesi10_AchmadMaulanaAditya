# DS_Sesi10_AchmadMaulanaAditya

Web Automation using WebdriverIO and Allure reporting for saucedemo.com website

### How to run this automation

1. install dependencies using
   `npm install`
2. run command for Web Automation
   `npm run wdio`

### Allure Report

1. Run command `npm run generate-allure` to generate the report
2. Run command `npm run open-allure` to view the the report
